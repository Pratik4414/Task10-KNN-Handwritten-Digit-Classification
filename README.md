# KNN Handwritten Digit Classification

## Objective
To classify handwritten digits using K-Nearest Neighbors (KNN).

## Dataset
Sklearn Digits Dataset (load_digits)

## Steps Performed
- Data loading and visualization
- Feature scaling
- KNN training
- K tuning
- Accuracy vs K plot
- Confusion Matrix

## Tools Used
- Python
- Scikit-learn
- Matplotlib
- Seaborn

## Result
Best accuracy achieved by tuning K value.
